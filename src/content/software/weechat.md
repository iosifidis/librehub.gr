Το WeeChat είναι ο terminal IRC client για power users — όσους θέλουν να τρέχουν έναν IRC client σε remote server (screen/tmux) και να συνδέονται από οπουδήποτε. Η scripting ευελιξία του και το relay σύστημα το κάνουν αγαπητό σε developers και sysadmins.

**Κύρια Χαρακτηριστικά:**
- **Terminal-based:** Τρέχει σε κάθε terminal χωρίς GUI.
- **Multi-protocol μέσω Plugins:** IRC native + Matrix, Slack, XMPP, Discord μέσω plugins.
- **Scripting:** Python, Perl, Ruby, Lua, Tcl scripts — εκατοντάδες διαθέσιμα scripts.
- **Relay:** Σύνδεση mobile/web clients στον WeeChat που τρέχει σε server.
- **Buflist:** Οργάνωση καναλιών και conversations σε sidebar.
- **Filter & Highlight:** Ισχυρά εργαλεία φιλτραρίσματος μηνυμάτων.
- **WeeChat-Android / Glowing Bear:** Mobile/web clients μέσω relay.

**Πλεονεκτήματα:**
- Ιδανικό για SSH-based workflow — τρέχει παντού με terminal.
- Ενεργή ανάπτυξη με τακτικές ενημερώσεις.
- Διαθέσιμο για Linux, macOS, Windows (WSL).
