Το Krut Computer Recorder είναι ένα απλό εργαλείο screen recording γραμμένο εξ ολοκλήρου σε Java — γεγονός που το κάνει cross-platform από τη φύση του. Σχεδιάστηκε κυρίως για την ανάγκη δημιουργίας εκπαιδευτικών βίντεο (screencasts) με ήχο.

> **Σημείωση:** Το Krut δεν αναπτύσσεται ενεργά. Για σύγχρονη εγγραφή οθόνης συνιστάται το **OBS Studio** (πλούσιο σε δυνατότητες) ή το **SimpleScreenRecorder** (Linux, ελαφρύ).

**Κύρια Χαρακτηριστικά:**
- **Επιλογή Περιοχής:** Εγγραφή ολόκληρης της οθόνης ή επιλεγμένης περιοχής.
- **Εγγραφή Ήχου:** Ταυτόχρονη εγγραφή ήχου από μικρόφωνο ή system audio.
- **Picture-in-Picture:** Εμφάνιση webcam μέσα στο screen recording.
- **QuickTime Export:** Εξαγωγή σε MOV format.
- **Cross-platform:** Λειτουργεί σε κάθε σύστημα με Java Runtime.

**Πλεονεκτήματα:**
- Απλότατο στη χρήση — ελάχιστη καμπύλη εκμάθησης.
- Cross-platform λόγω Java — ίδια εμπειρία σε Windows, macOS, Linux.
- Μηδενικές εξαρτήσεις εκτός από Java Runtime.
