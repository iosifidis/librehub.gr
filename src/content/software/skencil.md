Το Skencil (πρώην Sketch) είναι ένα από τα παλαιότερα διανυσματικά προγράμματα για Linux, γραμμένο εξ ολοκλήρου σε Python. Αν και δεν αναπτύσσεται πλέον ενεργά, παραμένει λειτουργικό για απλές διανυσματικές εργασίες σε Linux περιβάλλοντα.

> **Σημείωση:** Το Skencil δεν αναπτύσσεται ενεργά. Για νέα projects, το **Inkscape** είναι η συνιστώμενη επιλογή — πλουσιότερο, ενεργά συντηρούμενο και cross-platform.

**Κύρια Χαρακτηριστικά:**
- **Bezier Καμπύλες:** Δημιουργία και επεξεργασία διανυσματικών μονοπατιών.
- **Εργαλεία Σχημάτων:** Ορθογώνια, ελλείψεις, πολύγωνα, αστέρια.
- **Κείμενο:** Εισαγωγή και μορφοποίηση κειμένου.
- **PostScript Export:** Εξαγωγή σε PostScript και EPS μορφές.
- **Python Scripting:** Επέκταση λειτουργιών μέσω Python scripts.
- **Layers:** Βασική υποστήριξη επιπέδων.

**Πλεονεκτήματα:**
- Εξαιρετικά ελαφρύ — ιδανικό για παλαιότερα συστήματα Linux.
- Γραμμένο σε Python — εύκολη επέκταση για προγραμματιστές.
- Ιστορική σημασία ως ένα από τα πρώτα Linux vector editors.
