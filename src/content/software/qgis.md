Το QGIS είναι η κορυφαία open-source εναλλακτική του ArcGIS, κατάλληλη για χαρτογράφηση, χωρική ανάλυση, και διαχείριση δεδομένων GIS.

**Βασικές Λειτουργίες:**

- **Vector Data (Shapefile, GeoJSON, KML, GML, DXF, DWG, PostgreSQL/PostGIS, Spatialite, GeoPackage, WFS).**
- **Raster Data (GeoTIFF, NetCDF, ECW, JPEG2000, WMS, WMTS).**
- **Coordinate Reference Systems (CRS):** 8000+ προβολικά συστήματα (EPSG). Δυναμική αναπροβολή (on-the-fly reprojection).
- **Digitizing (Επεξεργασία GIS):** Δημιουργία και τροποποίηση vector layers (points, lines, polygons). Snap, Node tool, tracing.
- **Analysis Tools (Περίπου 300):** Buffer, Intersect, Clip, Dissolve, Merge, Difference, Area/length calculation, Nearest neighbor, Voronoi, Delaunay, Heatmap (Kernel Density), Slope, Aspect, Hillshade, Contour.
- **Processing Framework (SAGA, GRASS, GDAL, OTB):** Πρόσβαση σε χιλιάδες αλγόριθμους.
- **Cartography (Παραγωγή χάρτη):** Layout manager (προσθήκη scale bar, north arrow, legend, title). Export σε PDF, SVG, PNG.
- **3D View (Νέο):** Τρισδιάστατη απεικόνιση (3D map view) με terrain shading.
- **Plugins (500+):** Semi-Automatic Classification (SCP), QuickOSM (OpenStreetMap), Profile Tool, TimeManager, Google Satellite.
- **Python Console (PyQGIS):** Πλήρης σύνδεση με QGIS API, automation, scripting.
- **Πλατφόρμες:** Windows, macOS, Linux (official installers and repositories).

> **Σημείωση:** Χρησιμοποιείται από πανεπιστήμια (γεωγραφία, περιβαλλοντικές επιστήμες), κυβερνητικούς φορείς (Δήμοι, Υπουργεία), και εταιρείες (αγροτική παραγωγή, κατασκευές). Υποστηρίζει plugins (500+) και συνεργάζεται με PostGIS (βάση γεωχωρικών δεδομένων).
