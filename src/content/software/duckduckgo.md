Η DuckDuckGo είναι η πιο γνωστή εναλλακτική στη Google Search, με εκατοντάδες εκατομμύρια αναζητήσεις ημερησίως. Η βασική διαφορά είναι θεμελιώδης: η Google χτίζει λεπτομερές προφίλ για κάθε χρήστη (αναζητήσεις, τοποθεσία, συμπεριφορά) για στοχευμένες διαφημίσεις — η DuckDuckGo αγνοεί εντελώς ποιος κάνει την αναζήτηση.

**Κύρια Χαρακτηριστικά:**
- **Μηδενική Καταγραφή:** Δεν αποθηκεύονται αναζητήσεις, IP, User Agent ή ιστορικό συνεδριών.
- **!Bangs:** Πάνω από 13.000 συντομεύσεις για άμεση αναζήτηση σε άλλους ιστότοπους (π.χ. `!w` για Wikipedia, `!yt` για YouTube, `!gh` για GitHub).
- **Instant Answers:** Άμεσες απαντήσεις για μαθηματικά, μετατροπές, ορισμούς, καιρό κ.ά. χωρίς να χρειαστεί να επισκεφτείτε άλλη σελίδα.
- **Onion Site:** Διαθέσιμο και ως .onion υπηρεσία για χρήστες Tor.
- **DuckDuckGo App/Extension:** Ενσωματωμένος αποκλεισμός trackers και βαθμολόγηση ιδιωτικότητας για κάθε ιστότοπο.

**Πλεονεκτήματα:**
- Δεν δημιουργεί «filter bubble» — τα αποτελέσματα δεν εξατομικεύονται βάσει ιστορικού.
- Διαφημίσεις βάσει λέξης-κλειδιού (όχι προφίλ) — κερδοφόρα χωρίς παρακολούθηση.
- Διαθέσιμη ως προεπιλεγμένη μηχανή αναζήτησης σε Firefox, Tor Browser, και πολλούς άλλους.
