Το Linphone είναι ένα από τα πιο δημοφιλή και ώριμα softphones ανοιχτού κώδικα, διαθέσιμο σε όλες σχεδόν τις πλατφόρμες. Είναι μια πλήρης υλοποίηση του πρωτοκόλλου SIP (Session Initiation Protocol) και περιλαμβάνει τη δική του υποδομή (Flexisip server) για όσους θέλουν να τρέξουν το δικό τους VoIP δίκτυο.

Πλεονεκτήματα:
- **Cross-Platform**: Διαθέσιμο ως εφαρμογή για Windows, macOS, Linux, Android, iOS, καθώς και SDK για ενσωμάτωση σε web εφαρμογές.
- **Κρυπτογράφηση Άκρου-προς-Άκρο**: Υποστηρίζει SRTP (Secure Real-time Transport Protocol) και ZRTP για κρυπτογράφηση της ροής φωνής/βίντεο, εξασφαλίζοντας ότι κανένας ενδιάμεσος δεν μπορεί να υποκλέψει.
- **Πλούσιο UI**: Σύγχρονο γραφικό περιβάλλον με λίστα επαφών, ιστορικό κλήσεων, φωνητικές υπενθυμίσεις (voice mail) και δυνατότητα γραπτών μηνυμάτων (SIP SIMPLE).
- **Προηγμένη Διαχείριση Κλήσεων**: Κλήσεις αναμονής (hold), μεταφορά κλήσεων (transfer), κλήσεις αναμονής (waiting), τριμερείς διασκέψεις (conference).
- **Αποκεντρωμένο**: Μπορείτε να τρέξετε τον δικό σας SIP server (π.χ. Flexisip, Asterisk, FreeSWITCH) ή να χρησιμοποιήσετε δημόσιους SIP providers.
