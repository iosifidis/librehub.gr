Το KompoZer είναι ένας WYSIWYG (What You See Is What You Get) HTML editor, σχεδιασμένος για χρήστες που θέλουν να δημιουργούν ιστοσελίδες χωρίς να γράφουν κώδικα. Υποστηρίζει οπτική επεξεργασία (όπως το Dreamweaver σε λειτουργία design), CSS editor, ενσωματωμένο FTP upload, και διαχείριση μορφής κειμένου (bold, italics, εικόνες, πίνακες) σε περιβάλλον που θυμίζει word processor.

**Βασικές Λειτουργίες (για την εποχή του):**

- **WYSIWYG Editing:** Σύρετε κείμενο, εικόνες, πίνακες. Αλλάξτε χρώματα, γραμματοσειρές.
- **Dual View (Code / Preview):** Μπορείτε να μεταβείτε σε κώδικα και να τον επεξεργαστείτε χειροκίνητα.
- **CSS Editor (Built-in):** Δημιουργία CSS styles (προκαθορισμένα layouts).
- **FTP/SFTP Manager:** Διαχείριση πολλαπλών accounts, upload με ένα κλικ.
- **Form controls, tables, images, links** (διαδραστικά).
- **Table layout & design (όπως τα παλιά sites).**

**Σημεία προσοχής (2026):**

- **Εγκαταλελειμμένο project** (τελευταία έκδοση KompoZer 0.8 beta 3, 2010). Δεν υποστηρίζει HTML5, CSS3, JavaScript frameworks, responsive design.
- Λειτουργεί μόνο σε Windows (Linux μέσω Wine, deprecated).
- Δεν υποστηρίζει dark mode, code linting, live preview.

> **Σημείωση:** Το KompoZer **δεν συνιστάται** για επαγγελματική χρήση το 2026. Για απλή WYSIWYG editing, χρησιμοποιήστε **BlueGriffon** (νεότερο, βασισμένο σε Firefox rendering engine) ή **VS Code** με extensions (Live Server, HTML preview). Για CMS (WordPress), χρησιμοποιήστε block editor (Gutenberg).
