Το Leon είναι η open source εναλλακτική του Alexa και Google Assistant για τους χρήστες που θέλουν έναν έξυπνο βοηθό — αλλά χωρίς να στέλνουν τις φωνητικές εντολές τους στην Amazon ή Google. Τρέχει αποκλειστικά local (ή σε δικό σας server).

**Κύρια Χαρακτηριστικά:**
- **NLP:** Natural Language Processing για κατανόηση εντολών σε φυσική γλώσσα.
- **STT/TTS:** Speech-to-Text και Text-to-Speech για φωνητική αλληλεπίδραση.
- **Modules:** Python/Node.js modules για επέκταση δυνατοτήτων.
- **Skills:** Έτοιμα skills για web scraping, email, calendar, todo, IoT.
- **Multi-language:** Αγγλικά, Γαλλικά και άλλες γλώσσες.
- **API:** HTTP API για ενσωμάτωση με άλλες εφαρμογές.

**Πλεονεκτήματα:**
- 100% local — μηδενική κοινοποίηση δεδομένων.
- Επεκτάσιμο με custom modules σε Python/Node.js.
- Ενεργή ανάπτυξη με τακτικές ενημερώσεις.
- Διαθέσιμο για Linux, macOS, Windows (Docker).
