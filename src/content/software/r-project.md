Η R (όχι "R Project") είναι η γλώσσα της στατιστικής. Αποτελεί πρότυπο σε μεταπτυχιακά προγράμματα και ερευνητικά εργαστήρια.

**Βασικές Λειτουργίες (Core plus CRAN):**

- **Base R (δωρεάν):** Data frames, vectors, matrices, list, basic stats (mean, sd, t-test, ANOVA), plotting (plot, hist, boxplot).
- **CRAN Task Views (οργανωμένα):** Bayesian (rstan), Spatial (sf, sp), Machine Learning (caret, tidymodels), Time Series (forecast), Psychometrics (lavaan), Econometrics (plm), Genomics (Bioconductor).
- **Graphics:** `ggplot2` (grammar of graphics – industry standard).
- **Reporting (R Markdown, Quarto):** Δημιουργία αναφορών (HTML, PDF, Word) με ενσωματωμένο κώδικα R.
- **RStudio IDE (by Posit, not open source but free) ή RStudio Server (open-source version).**

> **Σημείωση:** Το R είναι απαραίτητο για τμήματα Στατιστικής, Ψυχολογίας (μεθοδολογία), Οικονομικών (econometrics), Βιοπληροφορικής, και άλλων θετικών επιστημών. Για business intelligence (dashboards), η μετάβαση σε Python (Pandas, Plotly) είναι συνηθισμένη, αλλά η στατιστική ισχύς του R παραμένει ασυναγώνιστη.
