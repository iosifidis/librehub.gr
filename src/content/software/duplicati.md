Το Duplicati είναι το «set it and forget it» backup tool: ρυθμίζετε τι θέλετε να backup-άρετε, πού και πότε, και αναλαμβάνει αυτόματα. Υποστηρίζει εξαιρετικά μεγάλο αριθμό backend destinations — αν έχετε λογαριασμό σε οποιονδήποτε cloud πάροχο, το Duplicati πιθανότατα τον υποστηρίζει.

**Υποστηριζόμενα Backends:**
Amazon S3, Backblaze B2, OneDrive, Google Drive, Dropbox, Box, SFTP, FTP, WebDAV, Azure Blob, Rackspace Cloud Files, Mega.nz, Jottacloud και πολλά άλλα.

**Κύρια Χαρακτηριστικά:**
- **Web UI:** Πλήρης διαχείριση μέσω browser — λειτουργεί και ως server service.
- **AES-256 Encryption:** Κρυπτογράφηση πριν την αποστολή — zero-knowledge για τον cloud πάροχο.
- **Incremental Backups:** Αποθήκευση μόνο των αλλαγών για εξοικονόμηση χώρου και χρόνου.
- **Deduplication:** Αποφυγή αποθήκευσης ίδιων δεδομένων πολλές φορές.
- **Scheduling:** Αυτόματο χρονοπρόγραμμα (ωριαία, ημερήσια, εβδομαδιαία).
- **Versioning:** Διατήρηση πολλαπλών εκδόσεων αρχείων με configurable retention.

**Πλεονεκτήματα:**
- Η μεγαλύτερη υποστήριξη cloud backends από κάθε ανοιχτού κώδικα backup εργαλείο.
- Λειτουργεί ως background service — δεν χρειάζεται να θυμάστε να το τρέξετε.
