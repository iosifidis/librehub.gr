Το GParted (GNOME Partition Editor) είναι το industry standard για γραφική διαχείριση κατατμήσεων σε Linux. Χρησιμοποιείται από sysadmins, τεχνικούς υπολογιστών και power users παγκοσμίως — και περιλαμβάνεται σε σχεδόν κάθε Linux rescue/live environment.

**Υποστηριζόμενα Filesystems:**
ext2, ext3, ext4, FAT16, FAT32, exFAT, NTFS, Btrfs, XFS, JFS, ReiserFS, f2fs, LVM, LUKS και άλλα.

**Κύρια Χαρακτηριστικά:**
- **Resize/Move:** Αλλαγή μεγέθους και μετακίνηση κατατμήσεων χωρίς απώλεια δεδομένων.
- **Create/Delete:** Δημιουργία νέων κατατμήσεων σε unallocated χώρο.
- **Format:** Μορφοποίηση κατατμήσεων σε οποιοδήποτε filesystem.
- **Check:** Έλεγχος ακεραιότητας filesystem.
- **Copy/Paste:** Αντιγραφή ολόκληρης κατάτμησης.
- **UUID & Label:** Αλλαγή UUID και ετικέτας κατάτμησης.
- **GParted Live:** Bootable ISO για εργασία σε Windows κατατμήσεις εκτός λειτουργίας.

**Πλεονεκτήματα:**
- Το ασφαλέστερο εργαλείο για resize NTFS κατατμήσεων (Windows) από Linux.
- Ιδανικό για προετοιμασία δίσκων για dual-boot εγκατάσταση.
- Διαθέσιμο για Linux και ως GParted Live ISO (bootable).
