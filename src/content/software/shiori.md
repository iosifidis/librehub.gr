Το Shiori (日本語: 栞, σελιδοδείκτης) είναι ένα command-line-first εργαλείο σελιδοδεικτών με web UI, γραμμένο σε Go για μέγιστη απόδοση και φορητότητα. Η μεγάλη του δύναμη: αποθηκεύει ολόκληρη την σελίδα τοπικά, όχι μόνο τον σύνδεσμο.

**Κύρια Χαρακτηριστικά:**
- **Offline Αρχειοθέτηση:** Αποθήκευση κάθε σελίδας ως HTML και EPUB για ανάγνωση χωρίς σύνδεση.
- **Reader Mode:** Καθαρή, αποσπαστική εμφάνιση άρθρων (αφαίρεση διαφημίσεων και πλευρικών στηλών).
- **Tag Οργάνωση:** Οργάνωση σελιδοδεικτών με tags.
- **CLI Interface:** Πλήρης διαχείριση από terminal — ιδανικό για scripting και αυτοματισμό.
- **Web UI:** Απλή, γρήγορη διεπαφή για περιήγηση και αναζήτηση.
- **REST API:** Ενσωμάτωση με άλλα εργαλεία και αυτοματισμούς.
- **Import:** Εισαγωγή από Netscape bookmark files.

**Πλεονεκτήματα:**
- Single binary — μηδενικές εξαρτήσεις, τρέχει οπουδήποτε.
- Ελάχιστες απαιτήσεις πόρων — ιδανικό για Raspberry Pi ή VPS.
- Ο συνδυασμός CLI + Web UI το κάνει εξαιρετικά ευέλικτο.
- Διαθέσιμο για Linux, macOS, Windows, Docker.
