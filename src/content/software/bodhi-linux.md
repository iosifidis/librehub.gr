Το Bodhi Linux ξεχωρίζει από άλλες ελαφριές διανομές με δύο τρόπους: το Moksha desktop environment έχει μια σχεδόν ανατολίτικη αισθητική (το όνομα «Bodhi» προέρχεται από το σανσκριτικό λέξη για «φώτιση»), και η φιλοσοφία του είναι minimalist — έρχεται με ελάχιστες προεγκατεστημένες εφαρμογές, αφήνοντας τον χρήστη να επιλέξει.

**Moksha Desktop:**
- Compositor με εντυπωσιακά γραφικά εφέ παρά τις ελάχιστες απαιτήσεις.
- Highly configurable — από minimal έως πλούσιο σε gadgets.
- Virtual desktops, shelves, και modules για κάθε λειτουργία.

**Εκδόσεις:**
- **Standard:** Για σύγχρονο 64-bit υλικό.
- **Legacy:** Για παλαιό 32-bit υλικό (υπολογιστές πριν το 2007).
- **AppPack:** Με περισσότερες προεγκατεστημένες εφαρμογές.
- **HWE:** Με νεότερο kernel για πρόσφατο υλικό.

**Πλεονεκτήματα:**
- Από τις λίγες διανομές που συνεχίζουν να υποστηρίζουν 32-bit αρχιτεκτονική.
- Βασίζεται στο Ubuntu LTS — πρόσβαση σε τεράστιο repository.
- Ιδανικό για αναζωογόνηση υπολογιστών δεκαετίας 2000-2010.
