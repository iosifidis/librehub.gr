Το Debian είναι η «γιαγιά» των Linux διανομών — ένα από τα αρχαιότερα και πιο σεβαστά projects του ανοιχτού κώδικα, που ξεκίνησε το 1993. Αναπτύσσεται εξ ολοκλήρου από εθελοντές χωρίς εταιρική χρηματοδότηση και ακολουθεί αυστηρές αρχές ελεύθερου λογισμικού.

**Τρεις Εκδόσεις:**
- **Stable:** Εξαιρετικά δοκιμασμένο — ιδανικό για servers και production. Ενημερώνεται κάθε 2 χρόνια.
- **Testing:** Ισορροπία σύγχρονου λογισμικού και σταθερότητας — για desktop χρήση.
- **Unstable (Sid):** Rolling release — τελευταίο λογισμικό, για προχωρημένους.

**Κύρια Χαρακτηριστικά:**
- **APT Package Manager:** Το πρότυπο package management που ακολουθούν Ubuntu και παράγωγα.
- **60.000+ Packages:** Η μεγαλύτερη βιβλιοθήκη packages από κάθε διανομή.
- **Multi-Architecture:** x86, x86-64, ARM, MIPS, PowerPC, RISC-V, s390x και άλλες.
- **Pure Free Software:** Κυρίως ελεύθερο λογισμικό — non-free διαθέσιμο ως προαιρετικό.
- **Debian Social Contract:** Νομικά δεσμευτική δέσμευση στο ελεύθερο λογισμικό.

**Πλεονεκτήματα:**
- Η πιο αξιόπιστη επιλογή για servers — χρησιμοποιείται σε κρίσιμες υποδομές παγκοσμίως.
- Ανεξάρτητο από εταιρικά συμφέροντα — κοινοτικό project με διαφανή governance.
- Βάση για το μεγαλύτερο μέρος του Linux ecosystem.
