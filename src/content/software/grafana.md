Το Grafana είναι η de facto πλατφόρμα observability για DevOps teams — η εφαρμογή που μετατρέπει raw metrics από κάθε πηγή σε ακατάστατα δεδομένα σε κατανοητά, όμορφα dashboards σε πραγματικό χρόνο.

**Κύρια Χαρακτηριστικά:**
- **Data Sources:** 150+ connectors — Prometheus, InfluxDB, Loki, Elasticsearch, MySQL, PostgreSQL, CloudWatch, Azure Monitor κ.ά.
- **Panels:** Line charts, bar charts, heatmaps, stat panels, gauges, tables, geomaps.
- **Alerting:** Unified alerting με routing σε Slack, PagerDuty, email, webhook.
- **Annotations:** Σήμανση events στα γραφήματα για correlation.
- **Templating:** Variables για dynamic dashboards.
- **Loki Integration:** Log aggregation και correlation με metrics.
- **Dashboard Sharing:** Public ή authenticated dashboard sharing.

**Πλεονεκτήματα:**
- Εντελώς δωρεάν self-hosted — χωρίς per-host licensing.
- Τεράστια βιβλιοθήκη community dashboards στο grafana.com/dashboards.
- Διαθέσιμο μέσω Docker, binary, Kubernetes.
