Το Linkding (από το γερμανικό «Lesezeichen» = σελιδοδείκτης) είναι η quintessential self-hosted εφαρμογή σελιδοδεικτών: κάνει ένα πράγμα, και το κάνει εξαιρετικά καλά. Γρήγορο, ελαφρύ, χωρίς περιττά χαρακτηριστικά — ακριβώς όσα χρειάζεστε για να οργανώσετε τους συνδέσμους σας.

**Κύρια Χαρακτηριστικά:**
- **Tag-based Οργάνωση:** Ιεραρχική οργάνωση με tags και nested tags (π.χ. `dev/python`).
- **Full-text Search:** Αναζήτηση σε τίτλους, περιγραφές και URL σε πραγματικό χρόνο.
- **Αρχειοθέτηση Σελίδων:** Αποθήκευση snapshot της σελίδας (μέσω Singlefile) για μελλοντική αναφορά ακόμα και αν η σελίδα εξαφανιστεί.
- **Browser Extensions:** Extensions για Chrome και Firefox — αποθήκευση με ένα κλικ.
- **Bookmarklet:** Λειτουργεί και σε περιηγητές χωρίς extension.
- **REST API:** Πλήρες API για αυτοματισμό και ενσωμάτωση με άλλα εργαλεία.
- **Import/Export:** Εισαγωγή από Netscape bookmark files (συμβατό με όλους τους περιηγητές).
- **Sharing:** Προαιρετικά δημόσια προβολή σελιδοδεικτών.

**Πλεονεκτήματα:**
- Εξαιρετικά ελαφρύ — τρέχει άνετα σε Raspberry Pi ή $3 VPS.
- Docker deployment σε λεπτά.
- Καθαρή, γρήγορη διεπαφή χωρίς JavaScript bloat.
- Ενεργή ανάπτυξη με τακτικές ενημερώσεις.
