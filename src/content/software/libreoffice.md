Το LibreOffice είναι μια ισχυρή, δωρεάν και ανοιχτού κώδικα σουίτα γραφείου, που αποτελεί την κορυφαία εναλλακτική του Microsoft Office. Αναπτύσσεται από τη μη κερδοσκοπική οργάνωση The Document Foundation, με εκατοντάδες εθελοντές και συνεισφέροντες εταιρείες. Προσφέρει εξαιρετική συμβατότητα με τα σύγχρονα formats της Microsoft (.docx, .xlsx, .pptx).

**Περιλαμβάνει:**

- **Writer:** Κειμενογράφος (εναλλακτικό του Microsoft Word). Υποστηρίζει μακροεντολές, στυλ, αυτόματη δημιουργία ευρετηρίων και πινάκων περιεχομένων, καθώς και δυνατότητα εξαγωγής σε PDF.
- **Calc:** Υπολογιστικά φύλλα (εναλλακτικό του Microsoft Excel). Ισχυρές λειτουργίες, Pivot Tables, επεξεργαστής τύπων, δημιουργία γραφημάτων, υποστήριξη μακροεντολών.
- **Impress:** Παρουσιάσεις (εναλλακτικό του Microsoft PowerPoint). Εφέ μετάβασης, animation, σχεδίαση διαφανειών, υποστήριξη πολυμέσων.
- **Draw:** Διανυσματική σχεδίαση (αντίστοιχο του Visio, με λιγότερες δυνατότητες). Ιδανικό για διαγράμματα, ροής εργασιών, αφίσες.
- **Base:** Βάσεις δεδομένων (εναλλακτικό του Microsoft Access). Δημιουργία φορμών, αναφορών, σύνδεση με εξωτερικές βάσεις (MySQL, PostgreSQL).
- **Math:** Επεξεργαστής μαθηματικών εξισώσεων. Ενσωματώνεται σε Writer και άλλες εφαρμογές.

**Πλατφόρμες:** Windows, macOS, Linux (συμπεριλαμβανομένων των περισσότερων διανομών), διαθέσιμο και ως φορητή έκδοση (portable).
