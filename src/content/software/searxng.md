Το SearXNG (fork του Searx) είναι η πιο ριζοσπαστική εναλλακτική μηχανή αναζήτησης: αντί να βασίζεται σε έναν πάροχο, συγκεντρώνει αποτελέσματα από δεκάδες πηγές ταυτόχρονα — και μπορείτε να το εκτελείτε στον δικό σας server.

**Πώς Λειτουργεί:**
- Λαμβάνει το ερώτημά σας και το στέλνει ανώνυμα σε πολλαπλές μηχανές αναζήτησης.
- Συγκεντρώνει και αξιολογεί τα αποτελέσματα.
- Επιστρέφει ενοποιημένη λίστα — χωρίς cookies, χωρίς IP logging.

**Κύρια Χαρακτηριστικά:**
- **Πολλαπλές Πηγές:** Google, Bing, DuckDuckGo, Brave Search, Wikipedia, GitHub, YouTube, Reddit κ.ά.
- **Κατηγορίες:** Αναζήτηση σε web, images, videos, news, maps, science, files.
- **Self-hosted:** Εγκατάσταση στον δικό σας server — μηδενική εξάρτηση.
- **Public Instances:** Εκατοντάδες δημόσιες instances από την κοινότητα.
- **Preferences:** Ρύθμιση προτιμώμενων πηγών, γλωσσών και κατηγοριών.

**Πλεονεκτήματα:**
- Πλήρως ανοιχτός κώδικας — επαληθεύσιμη μηδενική παρακολούθηση.
- Ποικιλία αποτελεσμάτων από πολλαπλές πηγές ταυτόχρονα.
- Docker deployment σε λεπτά για self-hosting.
