Το Evince (από τα λατινικά «evincere» = αποδεικνύω) είναι ο επίσημος document viewer του GNOME — αντικαθιστά την ανάγκη για πολλαπλές εφαρμογές (PDF reader, DjVu viewer, Comic Book reader) με μία ενοποιημένη λύση.

**Υποστηριζόμενα Formats:**
- **PDF:** Πλήρης υποστήριξη — forms, annotations, signatures, layers.
- **PostScript/EPS:** Άμεση προβολή PS αρχείων.
- **DjVu:** Σαρωμένα έγγραφα υψηλής συμπίεσης.
- **DVI:** LaTeX output files.
- **XPS:** Microsoft XML Paper Specification.
- **Comic Books:** .cbr, .cbz, .cb7 archives.
- **TIFF:** Multi-page TIFF images.

**Κύρια Χαρακτηριστικά:**
- **Dual/Continuous View:** Μονές ή διπλές σελίδες, συνεχής κύλιση.
- **Annotations:** Προβολή (αλλά όχι δημιουργία) PDF annotations.
- **SyncTeX:** Synchronization με LaTeX editors (Gedit, Vim).
- **Presentation Mode:** Fullscreen παρουσίαση εγγράφων.
- **Print:** Άμεση εκτύπωση μέσω CUPS.

**Πλεονεκτήματα:**
- Πολύ ελαφρύτερο από Adobe Acrobat Reader.
- Sandbox-ed μέσω Flatpak για ασφάλεια.
- Διαθέσιμο για Linux (GNOME) — και ως Flatpak σε κάθε διανομή.
