Το WordPress (wordpress.org) είναι ένα από τα πιο επιτυχημένα open source projects στον κόσμο — τροφοδοτεί sites από μικρά blogs έως The New York Times, Sony Music, Microsoft News. Η διάκριση μεταξύ wordpress.org (ανοιχτό κώδικα, self-hosted) και wordpress.com (εμπορική πλατφόρμα) είναι κρίσιμη.

**WordPress.org vs WordPress.com:**
- **.org:** Ελεύθερο λογισμικό — εγκαθίσταται σε δικό σας server, πλήρης έλεγχος.
- **.com:** Εμπορική πλατφόρμα της Automattic — περιορισμένος έλεγχος, συνδρομές.

**Κύρια Χαρακτηριστικά:**
- **Block Editor (Gutenberg):** Drag & drop page builder με blocks.
- **Themes:** 10.000+ δωρεάν themes — από απλά blogs έως e-commerce.
- **Plugins:** 60.000+ plugins στο repository (WooCommerce, Yoast SEO, Elementor κ.ά.).
- **Multisite:** Διαχείριση δεκάδων sites από μία εγκατάσταση.
- **REST API:** Headless WordPress για modern web frameworks.
- **WooCommerce:** Plugin e-commerce που τροφοδοτεί 28% των online shops.

**Πλεονεκτήματα:**
- Η μεγαλύτερη κοινότητα CMS — απαντήσεις σε κάθε πρόβλημα.
- Χιλιάδες developers και agencies εξειδικευμένοι σε WordPress.
- Διαθέσιμο σε κάθε shared hosting με PHP/MySQL.
