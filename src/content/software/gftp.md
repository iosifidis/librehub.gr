Το gFTP είναι ο κλασικός FTP/SFTP client για GNOME/GTK περιβάλλοντα Linux. Υπάρχει από το 1998 και παραμένει σταθερό και αξιόπιστο εργαλείο για απλές μεταφορές αρχείων σε server.

**Κύρια Χαρακτηριστικά:**
- **Dual-pane UI:** Τοπικά αρχεία αριστερά, remote server δεξιά.
- **Multithreaded:** Ταυτόχρονες λήψεις/αποστολές χωρίς μπλοκάρισμα.
- **Bookmarks:** Αποθήκευση και γρήγορη πρόσβαση σε servers.
- **Recursive Transfer:** Αναδρομική μεταφορά ολόκληρων δέντρων φακέλων.
- **Auto-reconnect:** Αυτόματη επανασύνδεση σε περίπτωση αποσύνδεσης.
- **FTP Proxy:** Υποστήριξη FTP μέσω proxy server.
- **CLI Mode:** Χρήση χωρίς GUI για scripting.

**Πλεονεκτήματα:**
- Ελαφρύ και γρήγορο — ιδανικό για παλαιότερα συστήματα Linux.
- Διαθέσιμο για Linux/Unix (GTK2).
