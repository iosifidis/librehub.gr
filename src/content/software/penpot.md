Το Penpot (από την Kaleidos) είναι η ελπίδα για open-source UI/UX design.

**Βασικές Λειτουργίες:**

- **Vector Design:** Δημιουργία διανυσματικών σχημάτων (ορθογώνια, κύκλοι, μονοπάτια), boolean operations (union, subtract, intersect), pen tool (Bezier).
- **Components & Libraries (Design System):** Δημιουργία επαναχρησιμοποιήσιμων components (buttons, cards, input fields). Σύνδεση με libraries (shared).
- **Interactive Prototyping:** Σύνδεση frames (artboards) με interactions (on click, hover). Πλοήγηση σε προεπισκόπηση.
- **Grids & Constraints (Auto layout):** Layout με grids (columns, rows) και constraints (pin, stretch).
- **Real-time Collaboration:** Πολλοί χρήστες μπορούν να επεξεργάζονται ταυτόχρονα (με cursors, annotations).
- **Export (CSS, SVG, PNG, JPEG, JSON):** Οι προγραμματιστές μπορούν να αντιγράψουν CSS (position, size, colors) απευθείας ή να κατεβάσουν assets.
- **Themes & Color Styles.**
- **Self-hosted (Docker).**
- **Penpot Cloud (SaaS):** δωρεάν tier (1GB storage, unlimited projects, 3 editors).

> **Σημείωση:** Ιδανικό για ομάδες που θέλουν να ελέγχουν τα σχέδιά τους (self-hosted). Το Figma είναι πιο πλούσιο (plugins, community, variable fonts), αλλά το Penpot εξελίσσεται γρήγορα και καλύπτει 90% των αναγκών.
