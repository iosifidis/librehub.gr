Το GNU Bayonne είναι μια υποδομή (framework) για την ανάπτυξη εφαρμογών τηλεφωνίας και φωνητικής απόκρισης (IVR - Interactive Voice Response). Σε αντίθεση με ένα softphone, το Bayonne είναι ένας server που "ακούει" σε τηλεφωνικές γραμμές, απαντά σε κλήσεις, παίζει ηχογραφημένα μηνύματα, αναγνωρίζει πλήκτρα (DTMF), εκτελεί εντολές και συνδέει κλήσεις.

Τεχνικές δυνατότητες:
- **Επεκτάσιμη Αρχιτεκτονική Scripting**: Οι εφαρμογές τηλεφωνίας γράφονται είτε σε C++ είτε μέσω μιας γλώσσας scripting (Bayonne CCL) που επιτρέπει γρήγορη ανάπτυξη IVR σεναρίων.
- **Υποστήριξη Telephony Hardware & Drivers**: Μπορεί να λειτουργήσει με παλαιότερες κάρτες PSTN (μέσω του σχετικού driver), αλλά κυρίως με VoIP μέσω SIP, H.323, και Analog.
- **Ενσωμάτωση με Databases**: Μπορεί να κάνει αναζήτηση σε βάσεις δεδομένων (SQL) κατά τη διάρκεια μιας κλήσης, π.χ. να δώσει υπόλοιπο λογαριασμού ή να καταγράψει μια παραγγελία.
- **Διαχείριση Κλήσεων (Call Routing)**: Μπορεί να λειτουργεί ως ένα μεταγωγέας (switch), κατανέμοντας εισερχόμενες κλήσεις σε διαθέσιμους πράκτορες (call center).
- **Συμβατότητα**: Μέρος του οικοσυστήματος GNU Telecom, με διασύνδεση σε Asterisk και άλλα telephony engines.
