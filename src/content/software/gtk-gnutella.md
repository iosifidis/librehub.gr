Το gtk-gnutella (GTK Gnutella) είναι ένας από τους παλαιότερους και πιο σταθερούς clients για το Gnutella δίκτυο — ένα από τα πρώτα αποκεντρωμένα P2P δίκτυα που δημιουργήθηκαν ως εναλλακτικό στο κεντρικοποιημένο Napster.

**Κύρια Χαρακτηριστικά:**
- **Gnutella 0.6 Protocol:** Πλήρης υλοποίηση του σύγχρονου Gnutella πρωτοκόλλου.
- **DHT (Kademlia):** Κατανεμημένη αναζήτηση χωρίς κεντρικούς κόμβους.
- **GUESS:** Gnutella UDP Extension for Scalable Searches.
- **Partial File Sharing:** Κοινοποίηση μερικώς κατεβασμένων αρχείων.
- **UPnP/NAT-PMP:** Αυτόματη ρύθμιση router για καλύτερη συνδεσιμότητα.
- **Bandwidth Scheduling:** Έλεγχος χρήσης bandwidth ανά ώρα.

**Πλεονεκτήματα:**
- Πλήρως αποκεντρωμένο — κανένας κεντρικός server που μπορεί να κλείσει.
- Ενεργή ανάπτυξη παρά την ηλικία του project.
- Διαθέσιμο για Linux, macOS, Windows.
