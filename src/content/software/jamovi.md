Το Jamovi (αντίστοιχο του JASP) είναι ιδανικό για φοιτητές και ερευνητές που δεν θέλουν να γράφουν κώδικα R, αλλά χρειάζονται στατιστικές μεθόδους και όμορφη παρουσίαση.

**Υποστηριζόμενες Αναλύσεις (Modules):**

- **Descriptives (Περιγραφικά), Frequencies, Correlations (Pearson/Spearman), Reliability (Cronbach's α).**
- **T-tests (independent, paired), ANOVA (one-way, factorial, repeated measures, ANCOVA).**
- **Linear Regression (ομαλή, stepwise), Logistic Regression.**
- **Non-parametric tests (Mann-Whitney, Kruskal-Wallis).**
- **Factor Analysis (EFA), Principal Component Analysis (PCA).**
- **Meta-Analysis (μέσω module).**
- **Προσθήκη modules από library (jamovi library).**
- **Αποτελέσματα:** Interactive tables (filtering, sorting), APA style export (Word, PDF).

> **Σημείωση:** Το Jamovi είναι η καλύτερη εισαγωγή στη στατιστική (επίπεδο προπτυχιακού). Για προχωρημένες αναλύσεις (mixed models, Bayesian), το JASP (επίσης open source) είναι συγκρίσιμο, αλλά με διαφορετικό UI. Το Jamovi κερδίζει σε απλότητα (spreadsheet-like).
