Το DokuWiki λύνει ένα κοινό πρόβλημα: χρειάζεστε ένα wiki για τεκμηρίωση αλλά δεν θέλετε να στήσετε MySQL ή PostgreSQL. Το DokuWiki αποθηκεύει τα πάντα σε αρχεία κειμένου — εγκατάσταση σε λεπτά, backup με ένα `tar`, version control μέσω αρχείων.

**Κύρια Χαρακτηριστικά:**
- **File-based Storage:** Χωρίς βάση δεδομένων — όλα σε plain text files.
- **DokuWiki Syntax:** Απλή, ευανάγνωστη σύνταξη για μη-τεχνικούς χρήστες.
- **Version Control:** Ενσωματωμένο ιστορικό αλλαγών για κάθε σελίδα.
- **Access Control:** Λεπτομερής έλεγχος πρόσβασης ανά χρήστη/ομάδα/σελίδα.
- **Plugin System:** 1000+ plugins (WYSIWYG editor, diagrams, task lists, calendar).
- **Templates:** Προσαρμογή εμφάνισης με themes.
- **Export:** PDF, ODT export μέσω plugins.
- **Media Manager:** Ενσωματωμένη διαχείριση εικόνων και αρχείων.

**Πλεονεκτήματα:**
- Η ευκολότερη εγκατάσταση wiki — μόνο PHP, χωρίς database.
- Backup = αντιγραφή φακέλου. Migration = ίδιο.
- Εξαιρετικό για intranet documentation, lab notes, project wikis.
- Διαθέσιμο σε κάθε shared hosting με PHP.
