Το GlobaLeaks είναι η ανοιχτού κώδικα εναλλακτική λύση για οργανισμούς που θέλουν να δημιουργήσουν ασφαλή καναλία καταγγελιών — δημοσιογραφικά μέσα, ΜΚΟ, δημόσιες υπηρεσίες, εταιρείες. Χρησιμοποιείται από δεκάδες μέσα ενημέρωσης και οργανισμούς παγκοσμίως.

**Κύρια Χαρακτηριστικά:**
- **Tor Integration:** Υποβολή πληροφοριών αποκλειστικά μέσω .onion address για μέγιστη ανωνυμία.
- **End-to-End Encryption:** Κρυπτογράφηση όλων των υποβολών με κλειδιά του παραλήπτη.
- **Two-way Communication:** Ασφαλής επικοινωνία μεταξύ πηγής και παραλήπτη χωρίς αποκάλυψη ταυτότητας.
- **GDPR Compliance:** Αυτόματη διαγραφή δεδομένων, audit trails, data minimization.
- **Multi-tenant:** Ένας server για πολλαπλούς οργανισμούς.
- **Questionnaire Builder:** Προσαρμόσιμες φόρμες υποβολής.

**Πλεονεκτήματα:**
- Χρησιμοποιείται από The Guardian, Transparency International, Μέσα Ενημέρωσης ΕΕ.
- Αυτο-φιλοξενούμενο — πλήρης έλεγχος δεδομένων.
- Διαθέσιμο μέσω Docker για εύκολη εγκατάσταση.
