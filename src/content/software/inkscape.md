Το Inkscape είναι το πιο προηγμένο vector graphics editor ανοιχτού κώδικα, ιδανικό για logos, εικονογραφήσεις, διαγράμματα, χάρτες, UI/UX mockups, ακόμα και για σχεδίαση laser cutting. Χρησιμοποιεί ως native format το **SVG (Scalable Vector Graphics)**, που είναι ανοιχτό πρότυπο W3C, αλλά υποστηρίζει και πολλά άλλα.

**Βασικές Δυνατότητες:**

- **Drawing Tools:** Bezier curves (pen tool), pencil (freehand), calligraphy, shapes (rectangle, ellipse, polygon, star, spiral), text (flowed text, text on path).
- **Node Editing (ισχυρό):** Επεξεργασία μονοπατιών (path nodes), ένωση, διαφορά, intersection, exclusion, division.
- **Layers & Objects:** Layers (με δυνατότητα lock/hide), groups, clones (συνδεδεμένα αντίγραφα), patterns.
- **Filters (SVG filters):** Blur, drop shadow, bevel, color matrix, bump map, displacement maps (as SVG, live rendering).
- **Color Management:** RGB, HSL, CMYK (περιορισμένη υποστήριξη, αλλά μπορείτε να ορίσετε CMYK values), gradients (linear/radial), mesh gradients (premium feature).
- **Text Support:** Advanced typography (OpenType features, kerning, tracking, baseline shift), text on path, flowed text (frames).
- **File formats (import/export):**
    - **Import:** SVG, PDF (multipage), EPS, AI (Adobe Illustrator, up to version 9 or later with limitations), CDR (CorelDRAW, μέσω unoconv), DXF, PNG, JPEG.
    - **Export:** SVG (clean), PDF (vector), EPS, PNG, DXF, HPGL, POVRay, κ.ά.
- **Extensions (Python):** Υπάρχει μεγάλη συλλογή extensions για βελτιστοποίηση, render (3D), plotting, CNC, laser cut, etc.

**Inkscape vs Illustrator:**

| Χαρακτηριστικό | Inkscape | Adobe Illustrator |
|---|---|---|
| **Native format** | SVG (ανοιχτό) | AI (κλειστό) |
| **Mesh gradients** | ✅ (limited, experimental) | ✅ (πλήρης) |
| **Image tracing (bitmap → vector)** | ✅ (βασικό) | ✅ (ισχυρό) |
| **Typography (OpenType features)** | ✅ (πλήρης, με GUI) | ✅ |
| **CMYK workflow** | ⚠️ (βασική) | ✅ (επαγγελματική) |
| **Κόστος** | Δωρεάν | $22.99/μήνα |

> **Σημείωση:** Το Inkscape είναι η κορυφαία επιλογή για vector graphics σε Linux, αλλά και σε Windows/macOS. Δεν ταιριάζει για επαγγελματική προεκτύπωση (CMYK + spot colors), αλλά για ψηφιακή χρήση (logos, web illustrations, UI design) είναι άριστο.
