Το Hydrogen είναι χρήσιμο για δημιουργία drum tracks.

**Βασικές Λειτουργίες:**

- **Pattern Editor (Matrix):** 64 patterns (bars) με 16 pads (όργανα). Pattern μήκους έως 192 ticks.
- **Song Editor (Timeline):** Συνδυασμός patterns για δημιουργία πλήρους τραγουδιού.
- **Layers (Stackable samples):** Μέχρι 8 samples ανά όργανο (π.χ. snare top, snare bottom, room reverb).
- **Humanization (Swing, Random velocity, Random timing).**
- **Sample Library (SF2, SFZ, AIFF, WAV):** Built-in drum kits (acoustic, electronic, Latin, GM). Μπορείτε να εισάγετε δικά σας samples.
- **Effects (Built-in):** Low-pass, High-pass, Sample rate reducer, Delay, Reverb, Compressor, Gate.
- **MIDI input:** Εγγραφή drum από MIDI pad ή πληκτρολόγιο (mapable).
- **Export:** WAV, FLAC, OGG, MIDI.
- **Πλατφόρμες:** Windows, macOS, Linux.

> **Σημείωση:** Εάν χρειάζεστε πλήρες DAW (όχι μόνο drum tracks), προτιμήστε **Audacity** (επεξεργασία) + **Hydrogen** (rhythm). Για πλήρη παραγωγή (MIDI, automation), εξετάστε **LMMS** ή **Ardour**.
