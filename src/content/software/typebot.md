Το Typebot είναι μια πλατφόρμα δημιουργίας **διαδραστικών φορμών που μοιάζουν με συνομιλία (conversational forms)** ή **chatbots** με οπτικό builder (όπως το Landbot, αλλά open-source). Οι χρήστες αλληλεπιδρούν με μια συζήτηση (chat-like UI) απαντώντας σε μηνύματα (κείμενο, κουμπιά, εικόνες, επιλογές). Κάθε απάντηση μπορεί να οδηγήσει σε διαφορετικό μονοπάτι (conditionals), επιτρέποντας σύνθετα quizzes, lead generation, appointment booking, customer support flows.

**Βασικές Λειτουργίες (Builder):**

- **Visual Builder (blocks):** Σύρετε και αφήνετε blocks (Message, Image, Video, Embed, Choice (buttons), Input (text, email, phone, number, date, file upload), Set variable, Condition (if/then), Redirect, Webhook, Script.
- **Logic Jumps:** Διακλαδώσεις (branching) με βάση τις απαντήσεις του χρήστη ή custom variables.
- **Integration with external APIs:** Webhooks (POST δεδομένων σε CRM, Google Sheets, Slack, Discord, HTTP endpoint).
- **Themes & Customization:** Προσαρμογή χρωμάτων, fonts, chat bubble, background image.
- **Embedding (ενσωμάτωση):** Μέσω iframe σε ιστοσελίδα, ή ως pop-up widget (με προσαρμόσιμο trigger).
- **Responses Dashboard:** Αποθήκευση απαντήσεων, export σε JSON/CSV, analytics (completion rate, drop-off points).
- **Self-hosted (AGPL):** Μπορείτε να εγκαταστήσετε σε δικό σας server (Docker, Node.js). Δωρεάν και χωρίς περιορισμούς (απεριόριστες φόρμες/απαντήσεις). Cloud version (Typebot.io) has free tier.

**Typebot vs Typeform (SaaS):**

| Χαρακτηριστικό | Typebot (Self-hosted) | Typeform (Basic) |
|---|---|---|
| **Διεπαφή (UI)** | Conversational (chat-like) | Παραδοσιακή φόρμα (one question per page) |
| **Drag-and-drop builder** | ✅ (πολύ διαισθητικό) | ✅ (καλό) |
| **Logic jumps** | ✅ (ισχυρό) | ✅ (βασικό) |
| **File uploads** | ✅ (built-in) | ❌ (μόνο για συνδρομητές) |
| **Self-hosted** | ✅ (AGPL) | ❌ |
| **Cost (unlimited responses)** | Δωρεάν (hosting) | ~$25/μήνα |

> **Σημείωση:** Το Typebot είναι η **καλύτερη open-source εναλλακτική για conversational forms** (quizzes, lead gen, customer surveys). Κατάλληλο για εταιρείες που θέλουν να αυτοματοποιήσουν lead qualification (π.χ. "Which product fits you?") ή customer support pre-screening. Για απλές φόρμες επικοινωνίας, το Tally (free) μπορεί να είναι αρκετό.
