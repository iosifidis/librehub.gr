Το Jami (πρώην SFLphone και GNU Ring) είναι μια επαναστατική πλατφόρμα επικοινωνίας που λειτουργεί χωρίς κανέναν κεντρικό server (peer-to-peer). Αντί να εμπιστεύεται έναν τρίτο (π.χ. Zoom, WhatsApp), το Jami χρησιμοποιεί μια κατανεμημένη δομή όπου οι συσκευές επικοινωνούν απευθείας, εξασφαλίζοντας απόλυτη ιδιωτικότητα και ανθεκτικότητα στη λογοκρισία.

**Μοναδικά Χαρακτηριστικά:**

- **Αποκεντρωμένο (P2P):** Δεν υπάρχουν κεντρικοί servers που κρατούν μετα-δεδομένα, επαφές ή περιεχόμενο. Η δρομολόγηση γίνεται μέσω του κατανεμημένου δικτύου (Distributed Hash Table - DHT) του OpenDHT.
- **Κρυπτογράφηση άκρου-προς-άκρο:** Όλες οι επικοινωνίες (μηνύματα, ήχος, βίντεο, αρχεία) κρυπτογραφούνται με TLS και RTCP, με χρήση ανθεκτικών κρυπτογραφικών αλγορίθμων.
- **Χωρίς Λογαριασμούς:** Η ταυτότητά σας είναι το δημόσιο κλειδί (fingerprint). Το Jami χρησιμοποιεί το blockchain (Ethereum) για αποκεντρωμένο καθολικό χώρο ονομάτων, αλλά η βασική χρήση γίνεται μέσω QR codes ή κωδικών.
- **Πλήρης Λειτουργικότητα:** Ομαδικές συνομιλίες (group chats), κλήσεις φωνής και βίντεο, κοινή χρήση οθόνης, αποστολή αρχείων, δωμάτια συνομιλίας.
- **Πολλαπλές Πλατφόρμες:** Εφαρμογές για Windows, macOS, Linux, iOS, Android, καθώς και διανομή μέσω F-Droid και Linux repos.

**Πλεονεκτήματα:**
- Μηδενική εξάρτηση από server υποδομή — λειτουργεί ακόμα και αν η εταιρεία κλείσει.
- Δεν απαιτεί email, τηλέφωνο ή οποιαδήποτε προσωπική πληροφορία.
- Διαθέσιμο για Windows, macOS, Linux, Android, iOS.
- Επίσημο GNU project — μακροπρόθεσμη ανάπτυξη εγγυημένη.

> **Σημείωση:** Το Jami είναι μέρος του GNU Project, γεγονός που το καθιστά 100% ελεύθερο λογισμικό, με ισχυρή δέσμευση στις ελευθερίες των χρηστών.
