Το ProjectLibre είναι ένα επαγγελματικό λογισμικό διαχείρισης έργων που σχεδιάστηκε για να αντικαταστήσει το Microsoft Project, με το οποίο μοιράζεται πολύ παρόμοια διεπαφή χρήστη. Είναι η εξέλιξη του OpenProj και αποτελεί την πιο δημοφιλή desktop εφαρμογή ανοιχτού κώδικα για project management, με εκατομμύρια λήψεις.

**Βασικές Δυνατότητες:**

- **Συμβατότητα με Microsoft Project:** Ανοίγει και αποθηκεύει αρχεία .MPP, .MPX, .XML, επιτρέποντας απρόσκοπτη συνεργασία με χρήστες MS Project.
- **Gantt Charts:** Πλήρως διαδραστικά διαγράμματα Gantt, με ορόσημα, εξαρτήσεις εργασιών (FS, SS, FF, SF), lead/lag, critical path.
- **Διαχείριση Πόρων (Resource Management):** Ανάθεση πόρων (άνθρωποι, εξοπλισμός), διαχείριση κόστους (cost types: work, material, fixed), leveling πόρων, Resource Breakdown Structure (RBS).
- **Αναφορές:** Πίνακες εργασιών (task tables), χρήση πόρων (resource usage), αναφορές κόστους, export σε PDF, HTML, CSV.
- **Ημερολόγια:** Πολλαπλά ημερολόγια (project, resource, task) με υποστήριξη αργιών, βάρδιων, διαθεσιμότητας πόρων.
- **Διαχείριση Project Portfolio (PPM):** Δυνατότητα διαχείρισης πολλαπλών έργων ταυτόχρονα (συγχώνευση αρχείων).

**Πλατφόρμες & Εκδόσεις:**

- **ProjectLibre (Desktop - δωρεάν):** Εφαρμογή για Windows, macOS, Linux. Πλήρης λειτουργικότητα για μεμονωμένους project managers.
- **ProjectLibre Cloud (επί πληρωμή):** Web-based έκδοση για ομαδική συνεργασία, με δυνατότητα real-time editing, cloud storage (AWS), και αναφορές.

> **Σημείωση:** Το ProjectLibre έχει βραβευτεί ως "Best Open Source Software" από το InfoWorld και χρησιμοποιείται από οργανισμούς όπως η NASA, Boeing, Department of Defense (ΗΠΑ), καθώς και από χιλιάδες μικρομεσαίες επιχειρήσεις.
