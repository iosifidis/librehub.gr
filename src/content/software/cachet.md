Το Cachet ήταν παλαιότερα η πιο δημοφιλής open-source status page. Ωστόσο, το project είναι **εγκαταλελειμμένο** (inactive από το 2022). Δεν λαμβάνει ενημερώσεις ασφαλείας, και υπάρχουν αναφορές για σφάλματα με σύγχρονες εκδόσεις PHP.

**Λειτουργίες (παλαιότερες):**

- Components (υπηρεσίες) & Groups.
- Incidents (περιστατικά) με updates.
- Metric points (π.χ. uptime, response time) με API εισαγωγής.
- JSON API.
- Support for MySQL, PostgreSQL, SQLite.
- Email notifications.

> **Σημείωση:** **Μην χρησιμοποιήσετε το Cachet** για νέα projects. Επιλέξτε **Kener** (dynamic, active) ή **Uptime Kuma** (monitoring + status page) ή **cState** (static).
