Το TestDisk και το PhotoRec είναι τα εργαλεία που φτάνουν στα χέρια τεχνικών όταν «δεν υπάρχει ελπίδα». Αναπτύχθηκαν από τον Christophe Grenier και συμπεριλαμβάνονται σε κάθε σοβαρό Linux rescue environment.

**TestDisk — Ανάκτηση Κατατμήσεων:**
- Επιδιόρθωση κατεστραμμένου MBR ή GPT partition table.
- Ανάκτηση διαγραμμένων κατατμήσεων.
- Επαναφορά boot sector (FAT, NTFS, exFAT).
- Αντιγραφή αρχείων από μη προσβάσιμες κατατμήσεις.
- Υποστήριξη: FAT, NTFS, ext2/3/4, HFS+, ReiserFS και άλλα.

**PhotoRec — Ανάκτηση Αρχείων:**
- Σάρωση φυσικού δίσκου για γνωστά file signatures (magic bytes).
- Ανάκτηση 400+ τύπων αρχείων: JPEG, PNG, MP4, PDF, DOCX, ZIP κ.ά.
- Λειτουργεί ακόμα και σε φορμαρισμένα media.
- Ανεξάρτητο από filesystem — διαβάζει raw sectors.

**Πλεονεκτήματα:**
- Τα καλύτερα δωρεάν εργαλεία ανάκτησης δεδομένων — συχνά καλύτερα από εμπορικές λύσεις.
- Διαθέσιμα για Windows, macOS, Linux, DOS.
- Περιλαμβάνονται στο SystemRescue, DBAN, GParted Live και άλλα rescue environments.
